The dynamic-setup block defines all of the dynamically generated mathematical objects. It also specifies particular constraints on any variables in the system as well as other environmental settings, such as required tolerance for values to match. The tag has an attribute seed which is used to provide a default seed for the initial state of the problem that will be used to create the static version.
