Instruction elements can define parameters, variables, and expression terms. Parameters can be generated randomly or calculated in terms of previously defined parameters. Expression terms can be computed in terms of previously defined parameters or expressions using the defined variables. All of these elements have @name attribute. Parameter and expression names should start with a letter and can contain letters A-Za-z, digits 0-9, and the underscore. I have to check whether variable names have the same flexibility, but they might only allow letters.
