@random

Value is a string specifying how the value is randomly assigned.

random='discrete' means the value will be chosen from an finite sequence of equally spaced values. Uses the additional attributes @min, @max, @by, and @nonzero.

random='uniform' means the value will be chosen uniformly between a minimum and maximum value. Uses the additional attributes @min and @max.

Need to think about markup for choosing one or more values from an arbitrary list.
