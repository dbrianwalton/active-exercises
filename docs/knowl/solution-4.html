Solution 4.3.1.

Noticing that the expression \([%= innerFormula.toTeX() %]\) appears inside parentheses with a power, it makes sense to think of that as the inner function, defining \(g(x) = [%= innerFormula.toTeX() %]\text{.}\) The outer function describes what happens to that. If we imagined replacing the formula \([%= innerFormula.toTeX() %]\) with a box and then call that box our variable \(x\text{,}\) we find the outer function is given by \(f(x) = [%= outerFormula.toTeX() %]\text{.}\)
