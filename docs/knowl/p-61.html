Noticing that the expression \(\) appears inside parentheses with a power, it makes sense to think of that as the inner function, defining \(g(x) = \text{.}\) The outer function describes what happens to that. If we imagined replacing the formula \(\) with a box and then call that box our variable \(x\text{,}\) we find the outer function is given by \(f(x) = \text{.}\)
